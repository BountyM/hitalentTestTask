# API организационной структуры

Тестовое задание: реализация REST API для управления подразделениями и сотрудниками компании с иерархической структурой.

## Технологии

- **Go** 1.22+ (стандартная библиотека `net/http`)
- **PostgreSQL** 15
- **GORM** – ORM для работы с БД
- **goose** – миграции базы данных
- **Docker** + **docker-compose** – контейнеризация и оркестрация
- **slog** – структурированное логирование
- **testify** + **gomock** – юнит- и интеграционные тесты
- **Make** – автоматизация сборки и запуска



## Быстрый старт

### Требования

- **Docker** (версия 20.10+)
- **Docker Compose** (версия 2.0+)
- **Make** (опционально, но рекомендуется)

### Запуск одной командой

```bash
make all
```
Эта цель:

копирует .env.example в .env (если не существует)

собирает Docker‑образ приложения

запускает контейнеры (docker-compose up -d)

применяет миграции БД (автоматически при старте приложения)

приложение становится доступно на http://localhost:8080
